Remove an existing user account.

```shell script
nest login [username]
```

See [users.json](../configuration/users.json.md) for technical details.

## Arguments

 argument | description
 -------- | -----------
 username | A username

:::info
If `[username]` is missing, you will be prompted to enter it.
:::

## Examples

```shell script
nest login
nest login me
```
